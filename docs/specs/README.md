# OpenSpec Specifications

This directory contains OpenSpec (OpenAPI + AsyncAPI) specifications for DiskCortex.

## Files

- `openapi.yaml` - REST API specification (TBD by Metis)
- `asyncapi.yaml` - Event-driven API specification (if needed)

## Usage

These specs drive the TDD development process:
1. Define spec first (OpenSpec)
2. Write failing test (TDD Red)
3. Implement minimal code (TDD Green)
4. Refactor (TDD Refactor)
